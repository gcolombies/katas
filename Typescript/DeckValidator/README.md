# TS Kata — Deck Validator (squelette)

## Prérequis
- Node.js >= 20
- npm (ou pnpm/yarn)

## Installation
```bash
npm install
```

## Lancer les tests
```bash
npm test
```

Mode watch :
```bash
npm run test:watch
```

## Build TypeScript
```bash
npm run build
```

## Structure
- `src/` : code du kata (interfaces + TODO)
- `tests/` : tests Vitest (squelettes + TODO)

> Objectif : implémenter le validateur en TDD.
